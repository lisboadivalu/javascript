Variaveis

para declararmos variaveis, podemos fazer da seguinte maneira:

var ex;

ex = ""valor 1

var ex = "exemplo";

ainda posso declarar varias variaveis dentro de uma so linha de codigo e depois atribuir um valor

a = 1;

b = 2;

c = a + b;

ou:

a = 1; b = 2; c = a + b;

var x = "5" + 2 + 3; -> se voce utiliza as aspas no primeiro numero da variavel, os outros serao tratados como string tambem

var x = 5 + 2 + "3"; -> mas se voce poe as aspas no ultimo numero da variavel, somente o que esta contendo as aspas sera tratado como string
